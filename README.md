# Генератор паролей от OliveWizard

## Описание программы
Программа позволяет создавать уникальные пароли длиной от 3 до 30 символов с различными условиями:
+ Строчные или заглавные буквы
+ Цифры
+ Дефис и нижнее подчеркивание
+ Специальные символы

Дополнительно можно отметить обязательность появления хотя бы одного символа из выбранных категорий и исключить похожие друг на друга или сложночитаемые символы.

## Системные требования
### Windows
Версия ОС - Windows 10 (1607) и новее.

Версия из файла PassGenWinNet.7z занимает меньше места, но требует установки [.Net 9](https://dotnet.microsoft.com/en-us/download)

Версия из файла PassGenWin.7z занимает больше меньше места, но не требует установки дополнительных компонентов.

### MacOS
Версия ОС - Mac OS 13 и новее.

Файл PassGenMac.7z не требует дополнительных зависимостей.
